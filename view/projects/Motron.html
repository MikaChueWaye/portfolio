project: Motron

Motron est un jeu de type "rétro" réalisé dans le cadre de ma S.A.E. du semestre 1 de ma 2e année de B.U.T. Informatique.

détails

En équipe de 5, nous avons établi la méthode agile SCRUM dans le but de garder un contact avec notre client.

Le projet s'est déroulé en 2 court sprint de 2 semaines puis 2 de 3 semaines avec notre rendu final étant une présentation de l'application déployable sur le marché.

langages utilisés:

  • Java
  • Java FX
  • SQL
  • CSS
  • ACCUEIL
  • QUI SUIS-JE?
  • MES PROJETS
  • CONTACT
