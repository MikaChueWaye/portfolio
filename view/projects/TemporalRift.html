Qu'est ce qu'une Code Game Jam? :

Une Code Game Jam est un évènement où différentes équipes s'affrontent dans un tournoi pour développer un jeu suivant un thème.

Après les 36h alloués au développement du jeu, les jurys décident des vainqueurs.
