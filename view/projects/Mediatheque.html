project: Mediatheque JavaScript

Dans le cadre de mon projet en Web: JavaScript du 4éme semestre, j'ai réalisé en équipe de 2 un site de médiathèque


Lien: Médiathèque JavaScript

détails :

Ce site web est doté d'une base de donnée sur laquelle on stocke les données de cette médiathèque propose les fonctionnalités de base d'une bibliothèque numérique en tant qu'administrateur :

  • consulter les livres disponibles
  • consulter les utilisateurs
  • consulter les livres empruntés
  • consulter les détails des emprunts (qui a emprunté quel livre / quel livre est emprunté par qui)
  • emprunter un livre
  • remettre un livre
  • supprimer un utilisateur/livre/emprunt

langages utilisés :

  • HTML
  • CSS
  • JavaScript
  • SQL
  • ACCUEIL
  • QUI SUIS-JE?
  • MES PROJETS
  • CONTACT
